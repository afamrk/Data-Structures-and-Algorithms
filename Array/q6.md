# Question
Given an array of integers (positive and negative) find the largest continuous sum. 
Input:

large_cont_sum([1,2,-1,3,4,10,10,-10,-1])
Output:

29

# Answer
### method 1
```python
def large_cont_sum(arr):
    high = sum = arr[0]
    for value in arr[1:]:
        sum = max(sum+value, value)
        high = max(high, sum)
    return high
```

### method 2
```python
def large_cont_sum(arr):
    high = sum = arr[0]
    for value in arr[1:]:
        if value < 0:
            if high < sum:
                high = sum
            if sum + value < 0:
                sum = value if value > sum else sum
            else:
                sum += value
        else:
            if sum < 0:
                sum = value
            else:
                sum += value
    if sum > high:
        high = sum
    return high
```
